# Lifestyle & Health Habits Dashboard

## Project Overview
This project is an interactive **Dashboard** built with **Dash** to analyze lifestyle and health habits data collected from a survey. Users can explore the relationships between variables such as sleep, stress, diet, exercise, and overall health.  

Key features:
- Interactive charts and graphs
- Drill-down tables for deeper insights
- Filters for gender, age group, and occupation
- KPI cards for quick statistics

---

## Dataset
- **Source:** Internal survey data (`Dash CW-2.xlsx`)  
- **Variables include:** Age group, Gender, Occupation, Sleep hours, Exercise frequency, Screen time, Study/work hours, Diet quality, Stress level, Energy level, Overall health, Lifestyle satisfaction  

---

## Features
- **KPI Cards:** Summary statistics (count, mean, median, min, max) for selected variables  
- **Charts:**  
  - Bar charts for overall health by age group  
  - Pie charts for exercise frequency  
  - Line charts for sleep hours distribution  
- **Drill-down insights:** Select an occupation to see average sleep and stress levels  
- **Filters:** Gender, age group, and occupation filters across charts and tables
